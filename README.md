# HOTEL-Admin
A system based on all the necessities a hotel can have for users. An entry level of programming . This program has been developed using the Java programming language, applying fundamental principles of Object-Oriented Programming (OOP), such as encapsulation, inheritance, and polymorphism. The system is designed with a modular structure that enhances scalability, maintainability, and code reusability.

The implementation includes control structures, exception handling, and the use of collections to efficiently manage data. In addition, best programming practices are followed, such as separation of concerns through well-defined classes and methods, which improves code readability and organization.

Furthermore, input validation is prioritized to ensure data integrity and to prevent runtime errors.

From a performance perspective, the system has been designed with resource efficiency in mind, incorporating appropriate algorithms for data processing.
